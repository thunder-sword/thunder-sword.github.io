应用包名

·adb tcpip 5555 ：将手机5555端口打开（然后才能通过wifi连接设备），关机重启后5555端口会默认关闭 ·adb connect 【手机ip】 ：通过wifi连接设备，关机重启后ip地址会改变 ·adb disconnect 【手机ip】 ：断开wifi连接 ·adb shell ifconfig wlan0 ：如果设备连着wifi，可以查询局域网ip，同时也有设备ip *adb exec-out screencap -p > 1.png ：截图保存到电脑 【一些新版本和一些旧版本都没办法用这个命令，那就只能用两条命令替代了： adb shell screencap -p /sdcard/sc.png adb pull /sdcard/sc.png 】 *顺便一提，usb调试并不是仅仅插入usb接口时需要连接，而是在用wifi连接时也需要usb调试

使用浏览器打开谷歌网站代码（网址可改）： adb shell am start -a android.intent.action.VIEW -d http://www.google.cn/

手机拨号至10086（号码可改【不需要开锁】）： adb shell am start -a android.intent.action.CALL -d tel:10086 经实验，不能输入MMI码，开头是米键则只会识别一个，开头是井号则会产生很奇妙的bug（我的是先是出现拨打号码的界面（但是没有号码），卡死，然后桌面变成了这个图片） *还有一个很好玩的现象是，拨打字母并不会出错，而是会把字母换算成数字来拨打，经实验，a-c代表2，d-f代表3，以此类推，每三个字母代表一个同样的数字，然后到7时，有四个（p-s），最后到9时，又有四个（w-z）；这正好是9键拼音中的数字对应位置，其实拨号键下面也有；这样可以推出，除了拨号键中有的符号，输入其他符号都会无法识别（包括除+#*的其他符号）

打开地图并且定位到北京（位置可改）： adb shell am start -a android.intent.action.VIEW geo:0,0?q=beijing

adb可以启动手机里面的程序，想请教一下，能不能在命令行里给程序带参数，比如说，启动视频播放器，指定打开某个视频。【待理解】 adb shell am start -d file:///sdcard/123.mp4 -t video/mp4 -a android.intent.action.VIEW
