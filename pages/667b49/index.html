    总结：许多情况下，你的知识贮备量比智商有用多了，还是要多做题，多总结知识点，不然有些东西你咋也解不出来，更有些东西你在搜索引擎上根本搜不到（比如说代码的一些漏洞）

    # 题目整理：

    # Web

    # 1.web签到

    直接f12找到flag

    # 2.easyserialize

    打开之后可以看到源代码，并且知道了flag在flag.php中，大体思路就是利用函数action将flag.php爆出来，但是还需要将输入反序列化。

    复制代码，在自己环境跑一下（此处用菜鸟编译器，一样效果），得到file="flag.php"时的序列化字符串。

    得到结果：

    O:4:"Test":1:{s:6:"target";O:7:"GetFlag":1:{s:4:"file";N;}}
    1

    只需要将最后的N换成s:8:"flag.php"就可以将file复制为相应值。

    因此完整的payload如下：

    ?a=O:4:"Test":1:{s:6:"target";O:7:"GetFlag":1:{s:4:"file";s:8:"flag.php";}}

    ps：菜鸟在线编译器：https://www.runoob.com/try/runcode.php?filename=demo_intro&type=php

    # 3.ezPHP

    这道题在比赛中卡在一个点没过去，但是记录一下，以备下次（感觉这题就是拼底蕴了）

    打开页面可以看到源代码：

    第一个点要求原文不同，密文相同

    因为php漏洞，md5后的值虽然是字符串，但是两个等于号比较时可能会转换成整形，就是xexxxxx这种类型的字符串，会被认为是科学计数法的整数。

    因此，采取两个不同字符串但是md5值均为0exxxxxx类型的字符串即可，因为都会被认为是0，此处随便选两个。

    zs=byGcY&dog=sonZ7y

    然后第二个点要求原文与md5密文相同，原本思路是找一个0exxx类型并且md5值也为0exxxx类型的字符串，但是好不容易找到之后，还是不行。

    本地跑过之后，发现如果等号一侧为字符串时，两个数值不会隐式转换为整形，返回false：

    呃……这个我在搜索引擎搜索那么长时间都找不到，就是储备问题了(或者可以直接md5强碰撞？我还不会/(ㄒoㄒ)/~~)，多刷题！

    最后一个点要求输入变量f_l_a_g并且不能输入下划线或者%5f码，同时变量既要等于23333，而且不能完全是23333。

    实际上就是两个点了。

    第一个点绕过方法很多，可以使用空格或者点或者%5F（substr_count函数区分大小写），第二个点在末尾加%0a（%0a是换行符）。

    这个是在别人的博客中学到的，如下：

    ?f l a g=23333%0a ?f.l.a.g=23333%0a ?f%5Fl%5Fa%5Fg=23333%0a

    完整的payload如下：

    ?zs=byGcY&dog=sonZ7y&cat=0e215962017&f%20l%20a%20g=23333%0a

    # Crypto

    # 1.easy_substitution

    搜索substitution，原来是替换密码的意思，是一种比较简单的类摩斯密码的密码，直接利用工具解决，

    我用了一款在线工具（需要科.学上网）：

    https://planetcalc.com/8047/

    比赛结束后大佬有国内可访问的在线工具：

    https://quipqiup.com/

    总之得到明文了：

    BY THE ALMIGHTY GOD YOU CAN READ THIS MY FRIEND I AM IMPRESSED VERY WELL DONE YOUR SOLUTION KEY IS INDAFBMHHLBO THIS LITTLE CHALLENGE WAS NOT TOO HARD WAS IT

    所以key就为：

    flag{INDAFBMHHLBO}
