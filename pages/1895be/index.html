3下载安装中文版vim帮助文档

# 缘由：

在终端输入命令vimtutor -g zh可以进入一种中文的帮助文档，但是这个并不全面

可以说vim自带了三种帮助文档，三种还都不一样：

  • 终端输入：vimtutor
  • vim输入：:help
  • vim输入：:help user-manual //用户手册

除了第一种以外，下面两种帮助文档都是默认没有中文版的，需要自己下载安装，就记录一下安装方法吧。

下面两种的帮助文档是可以一起安装的，这也倒是挺方便的。

# 安装方法：

在根目录随便创建一个文件夹，如果不想碍眼的话，就创建.vim/doc这个文件夹树，因为加了.，所以之后ls会被默认隐藏，挺好。

访问网址http://vimcdoc.sourceforge.net/

在其中选择“Latest platform independent tarball,including an Linux/BSD installation script”下载

//如果找不到这个就直接下载linux的解压包文件，然后解压之后，将其中的文件都复制到你创立的那个文件夹中，如.vim/doc。

然后直接运行它的安装脚本即可bash ./vimcdoc.sh -i。

# 注意事项：

vim中文文档不会覆盖原英文文档，安装后vim默认使用中文文档。 若想使用英文文档，可在home/.vimrc中执行以下命令： :set helplang=en （即将cn改为en） 同理，使用以下命令可重新使用中文文档： set helplang=cn

# Ps：

我这是8.2版本的vim，vimcdoc也到了2.3.0，所以能直接在vim中看到正常的中文帮助文档，但是低版本的可能无法正常显示，这时候就需要一定配置：

注意: a. 如果无法显示中文, 在home/.vimrc中增加下面这句试试： set helplang=cn b. 帮助文件的文本是utf-8编码的, 如果想用vim直接查看, 需要在home/.vimrc中设置： set encoding=utf-8
