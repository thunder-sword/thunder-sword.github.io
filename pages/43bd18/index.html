模拟输入

adb shell input 【+】 tap 1000 1730 ：在指定坐标点击一下（1000，1730分别指x值和y值，目前的坐标是QQ的发送按钮的位置） text ：在输入框上输入string字符串，该方法只能输入ASCII码，不能输入UTF-8的字符（汉字不能输入） keyevent ：模拟键盘按一次 [touchscreen|touchpad|touchnavigation] tap ：模拟点击操作 [touchscreen|touchpad|touchnavigation] swipe [duration(ms)] ：向设备发送一个滑动的指令，并且可以选择设置滑动的时长。 这个指令乍一看不就是个滑动嘛……233333……其实还有一个特殊的操作，长按 { //滑动 adb shell input swipe 100 100 200 200 300 //从 100 100 经历300毫秒滑动到 200 200 //长按 adb shell input swipe 100 100 100 100 1000 //在 100 100 位置长按 1000毫秒 } trackball press：这个命令是模拟轨迹球发送点击命令，因为现在手机设备上没有轨迹球，然并卵…… trackball roll ：这个命令是模拟轨迹球发送滚动命令，同样，然并卵……
